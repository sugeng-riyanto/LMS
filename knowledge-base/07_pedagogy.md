# Pedagogical Framework

## 1. Flipped Classroom Structure (40 minutes STRICT)

### Phase 1: Entry Ticket & Hook (5 minutes)
- Project 1 question from pre-class quiz (most missed)
- MythBuster or real-world analogy to spark curiosity
- Example: "Why do astronauts float if gravity is 90% as strong in orbit?"

### Phase 2: Productive Struggle (20 minutes) - Classwork 40%
- Students work on Level 1 & 2 worksheet questions
- HARD RULE: "Ask 3 Before Me" - must consult 3 peers before asking teacher
- Teacher circulates, observes, does NOT lecture
- Differentiation: support scaffolds for struggling, extension for advanced

### Phase 3: CER Challenge (10 minutes)
- Level 3 worksheet question: real phenomenon
- Students write Claim, Evidence, Reasoning
- Teacher projects mark scheme, students peer-grade

### Phase 4: Wrap-up & Mistake Journal (5 minutes)
- Students record 1 mistake from today in Mistake Journal
- Teacher announces next week's pre-class material
- Peer-grading finalized, worksheets collected

## 2. CER Framework (Claim-Evidence-Reasoning)

### Template
    CLAIM: State what you believe is true (1 sentence)
    EVIDENCE: Provide data, observations, or facts (2-3 sentences)
    REASONING: Connect evidence to physics laws/principles (2-3 sentences)

### Example (Grade 12 - Astronaut weightlessness)
    CLAIM: Astronauts are not weightless; they are in continuous free-fall.
    EVIDENCE: ISS experiences 90% of Earth's gravity. ISS has high tangential velocity (~28,000 km/h).
    REASONING: By Newton's law of gravitation and circular motion, the curvature of ISS's fall matches Earth's curvature. No normal contact force exists, which we perceive as 'weightlessness'.

## 3. Derivation Without Calculus (HARD RULE)

### Allowed Methods
1. Geometry: Area under graphs (v-t graph area = displacement)
2. Algebra: Substitution and rearrangement of equations
3. Vector resolution: Component analysis

### Banned Methods
- Integration (integral sign)
- Differentiation (d/dx, derivative)
- Any calculus notation

### Example: Deriving v^2 = u^2 + 2as
    Step 1: Draw v-t graph (trapezium shape)
    Step 2: Area = displacement s = (u+v)/2 * t ... (Eq.1)
    Step 3: Gradient = acceleration a = (v-u)/t, so t = (v-u)/a ... (Eq.2)
    Step 4: Substitute Eq.2 into Eq.1:
            s = (u+v)/2 * (v-u)/a
            2as = (v+u)(v-u)
            v^2 = u^2 + 2as  QED

## 4. Class Rules (Non-Negotiable)

### Rule 1: "Ask 3 Before Me"
- Before raising hand to teacher, student MUST consult 3 peers
- Builds collaboration, reduces teacher dependency
- Teacher only intervenes after peer consultation fails

### Rule 2: "No Eraser"
- Mistakes are data, not shame
- Students cross out errors with single line, write correction beside
- Teacher assesses thinking process, not just final answer
- Builds metacognition and scientific honesty

### Rule 3: "Mistake Journal"
- Every student maintains small notebook
- After each class, record:
    * What was my mistake?
    * Why did I make it? (root cause)
    * What is the correct approach?
    * Which physics law did I violate?
- Reviewed monthly by teacher, contributes to Project grade (10%)

## 5. Bloom's Taxonomy Mapping per Worksheet Level

### Level 1: Sanity Check (10 min)
- Bloom: Remember, Understand
- Question types: Definition, recall, simple calculation
- Peer-graded using mark scheme

### Level 2: Productive Struggle (20 min)
- Bloom: Apply, Analyze
- Question types: Multi-step problem, HOTS, past paper, Mistake Hunter
- Intentional teacher error for students to find
- Derivation using geometry/algebra

### Level 3: CER Challenge (10 min)
- Bloom: Evaluate, Create
- Question types: Phenomenon explanation, experimental design critique
- Must use CER structure
- Real-world context (TKA/IGCSE/Cambridge style)
