## How to Generate Your Own

Run the demo seed script to populate the database with sample packages for all grades:

```bash
node scripts/seed-demo.js
```

Then visit:
- `/grades` to browse all grade packages
- `/grades/10/1` to view Grade 10, Week 1 package
- `/lesson-plan` to generate custom lesson plans
- `/syllabus` to plan and generate syllabus content

## Download Formats

From any package detail page (`/grades/[grade]/[week]`):
- **DOCX** — Official SHB lesson plan template with school logo
- **PDF** — Landscape A4, ready to print
- **MD** — Editable Markdown
- **QMD** — Quarto Markdown (renders to PDF)

From the Lesson Plan Generator (`/lesson-plan`):
- Same 4 formats + Preview before download
